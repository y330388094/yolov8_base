这个文件目录下的所有文件，就是定义我们模型中的一些组成构建，之后我们进行改进和优化，增加其它结构的时候都要在对应的文件下
面进行改动。

modules文件夹:
init .py: 表明此目录是Python包。
block.py: 包含定义神经网络中的基础块，如残差块或瓶颈块，
conv.py: 包含卷积层相关的实现。
head.py: 定义网络的头部，用于预测。
transformer.py:包含Transformer模型相关的实现
utils.py: 提供构建神经网络时可能用到的辅助函数。
init .py: 同样标记这个目录为Python包。
autobackend.py: 用于自动选择最优的计算后端。tasks.pv: 定义了使用神经网络完成的不同任务的流程，例如分类、检测或分割，所有的流程基本上都定义在这里，定义模型前向传播都在这里。
